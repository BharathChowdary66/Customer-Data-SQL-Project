# SQL Customer Shopping Behavior Analysis

This project includes SQL queries and insights generated from a customer 
shopping behavior dataset. It covers:

### ✔ SQL Queries
- Aggregations (SUM, AVG, COUNT)
- Joins (INNER, LEFT, RIGHT)
- Subqueries
- Views for analysis
- Query optimization using indexes

### ✔ Files Included
- `/datasets/customer_shopping_behavior.csv` – The raw dataset
- `/sql/quaries.sql` – All SQL questions & answers
- `/report/SQL Data Analysis Report on Customer Shopping Behavior.pdf` – Final PDF report

### ✔ Tools Used
- PostgreSQL (pgAdmin)
- SQL
